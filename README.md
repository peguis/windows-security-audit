Windows Security Audit
Objetivo

Projeto desenvolvido com o objetivo de praticar conceitos de Segurança da Informação e auditoria básica de sistemas Windows durante minha transição para a área de Tecnologia da Informação.

A proposta foi realizar uma análise prática de segurança em um ambiente Windows, verificando configurações, proteções ativas e boas práticas recomendadas para usuários finais.

Recomendações de Segurança

Com base na auditoria realizada, recomenda-se:

Manter o Windows atualizado
Utilizar autenticação multifator sempre que possível
Utilizar senhas fortes e exclusivas
Manter antivírus ativo e atualizado
Evitar clicar em links suspeitos
Revisar periodicamente programas instalados
